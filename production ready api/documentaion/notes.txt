Instructions section → Gives specific tasks the AI should perform.
Examples section → Shows examples of expected input/output.
Output format section → Specifies how the answer should be structured.

| Regex    | Meaning                                           | Example Match                 |       |      |
| -------- | ------------------------------------------------- | ----------------------------- | ----- | ---- |
| `.`      | Any character                                     | `a`, `7`, `@`                 |       |      |
| `\d`     | Digit                                             | `5`                           |       |      |
| `\D`     | Not a digit                                       | `A`                           |       |      |
| `\w`     | Word character                                    | `hello`, `_`, `7`             |       |      |
| `\W`     | Non-word character                                | `@`, `-`                      |       |      |
| `\s`     | Whitespace                                        | Space, tab, newline           |       |      |
| `\S`     | Non-whitespace                                    | `A`, `9`                      |       |      |
| `[abc]`  | One of `a`, `b`, or `c`                           | `b`                           |       |      |
| `[^abc]` | Anything except `a`, `b`, or `c`                  | `x`                           |       |      |
| `*`      | Zero or more                                      | `ab*` → `a`, `ab`, `abb`      |       |      |
| `+`      | One or more                                       | `ab+` → `ab`, `abb`           |       |      |
| `?`      | Zero or one (or lazy modifier after a quantifier) | `colou?r` → `color`, `colour` |       |      |
| `{n}`    | Exactly `n` repetitions                           | `\d{4}`                       |       |      |
| `{n,m}`  | Between `n` and `m` repetitions                   | `\d{2,5}`                     |       |      |
| `^`      | Start of string (or negation inside `[]`)         | `^Hello`                      |       |      |
| `$`      | End of string                                     | `world$`                      |       |      |
