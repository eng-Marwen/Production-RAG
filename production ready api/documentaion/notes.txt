Common prompt sections:

System section → Defines the AI's role, rules, and behavior.
User section → Contains the user's request or question.
Context section → Contains additional information (for example, retrieved RAG documents).
Instructions section → Gives specific tasks the AI should perform.
Examples section → Shows examples of expected input/output.
Output format section → Specifies how the answer should be structured.

Separators like --- or === are often used to visually distinguish these sections,
